# Changelog

## [v1.0.1] - 2026-01-15

### Added
- **Payment Management in Invoices**: Users can now view a list of registered payments directly within the Invoice Edit form.
- **Edit/Delete Payments**: Added functionality to edit details of existing payments or delete them. modifying the invoice balance and status automatically.
- **Changelog System**: Established this changelog to track future updates.

### Changed
- Refactored `getInvoice` to include payment history.
- Updated `PaymentDialog` to support editing mode.
